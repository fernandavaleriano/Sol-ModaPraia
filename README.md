## Sobre este projeto

O **Sol Moda Praia** é um site modelo criado para venda.  
A proposta deste repositório é apresentar um projeto visualmente forte, moderno e pronto para ser entregue a um cliente final com personalização sob demanda.

> **Importante:** este site **não está sendo disponibilizado como projeto gratuito para uso livre**.

## Grande diferencial: painel administrativo para o comprador gerenciar sozinho

Esse é um dos pontos mais fortes deste projeto.

O comprador não fica dependente de programador para fazer as alterações mais importantes da loja.  
Após a compra, ele recebe acesso a uma **página administrativa** onde pode gerenciar boa parte da operação de forma prática.

### No painel administrativo, o comprador pode:
- cadastrar novos produtos
- editar produtos existentes
- excluir produtos
- alterar **nome do produto**
- alterar **preço**
- alterar **categoria**
- alterar **descrição**
- definir **tamanhos disponíveis**
- adicionar **fotos do produto**
- ativar ou desativar produtos exibidos na loja
- visualizar **pedidos realizados**
- acompanhar **clientes cadastrados**
- consultar **estatísticas gerais da operação**
- atualizar o **status dos pedidos** no painel

### O que isso significa na prática
O comprador consegue manter a loja atualizada sozinho, sem precisar pedir ajuda técnica sempre que quiser:
- trocar preços
- ajustar informações dos produtos
- cadastrar novas peças
- organizar o catálogo
- acompanhar pedidos
- acompanhar clientes
- conduzir a operação com muito mais autonomia

Esse ponto agrega muito valor comercial, porque transforma o projeto em mais do que um site bonito: ele entrega também **controle de gestão**.
